    * Java also provides **JShell** (introduced in Java 9) for quick testing.
    * Open in command prompt/terminal:


      ```bash

      9 - 6       // Output: 3
      print(6)    // Error
      System.out.print(6)     // Correct way
      System.out.println("Hello World")
      ```
