  <Step>
    ## Why Use Else If?

    * Sometimes we need to check **multiple conditions**.
    * `if-else-if` ladder allows checking conditions one by one.
    * Once a **true condition** is found, its block executes â†’ remaining blocks are skipped.
  </Step>

  <Step>
    ## Syntax

    ```java
    if (condition1) {
        // executes if condition1 is true
    } else if (condition2) {
        // executes if condition2 is true
    } else if (condition3) {
        // executes if condition3 is true
    } else {
        // executes if all conditions are false
    }
    ```
  </Step>

  <Step>
    ## Example â€“ Find the Greatest of Three Numbers

    ```java
    public class Demo {
        public static void main(String[] args) {
            int x = 8;
            int y = 7;
            int z = 9;

            if (x > y && x > z) {
                System.out.println(x);
            } else if (y > z) {
                System.out.println(y);
            } else {
                System.out.println(z);
            }
        }
    }
    ```

    ðŸ‘‰ Output: **9**
  </Step>

  <Step>
    ## Flow Explanation

    1. Check if `x` is greater than both `y` and `z`.

       * If true â†’ print `x`.
    2. Else, check if `y` is greater than `z`.

       * If true â†’ print `y`.
    3. Else (remaining case) â†’ print `z`.
  </Step>

  <Step>
    ## Key Notes

    * Execution stops at the **first true condition**.
    * `else` block is optional.
    * Useful for **multi-branch decision making**.
  </Step>
</Steps>
